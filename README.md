# 🐨 Pontua Aí

> ✨ *"Venha se tornar um coala!"*

O **Pontua Aí** é uma plataforma educacional desenvolvida para **alunos e professores**, com o objetivo de democratizar o acesso ao aprendizado e tornar o estudo mais prático, organizado e eficiente.

---

## Nossa missão

Acreditamos que TODOS devem ter as mesmas oportunidades de aprendizado.
Por isso, criamos uma plataforma simples, intuitiva e poderosa para apoiar o ensino.

---

## Funcionalidades para alunos

Com o Pontua Aí, você pode:

* Responder questões por **matéria e conteúdo específico**
* Fazer redações com correção automática *(sim, o Pont corrige tudo)*
* Acompanhar sua evolução
* Praticar e se aprimorar em diversas áreas

---

## Funcionalidades para professores

A plataforma também oferece ferramentas completas para educadores:

* Gerar provas automaticamente em **PDF**
* Cadastrar questões personalizadas
* Acompanhar desempenho dos alunos
* Visualizar quantas pessoas responderam cada questão

## 🐨 Mascote

### Sobre o Pont

O **Pont** é o mascote oficial do Pontua Aí, representado por um coala — e essa escolha não foi por acaso.

Os coalas são animais conhecidos por serem calmos, pacientes e persistentes, características essenciais no processo de aprendizagem. Queremos que os estudantes se identifiquem com esses valores:

- Persistência: para não desistir, mesmo diante das dificuldades  
- Calma: para enfrentar os desafios com equilíbrio  
- Paciência: para entender que aprender é um processo contínuo  

Assim como o coala sobe uma árvore com tranquilidade e constância, acreditamos que o aprendizado acontece um passo de cada vez.

O Pont representa exatamente isso: um guia amigável que acompanha o estudante na sua evolução.

## Sobre o projeto

O **Pontua Aí** é uma aplicação web que centraliza ferramentas educacionais em um só lugar, evitando:

* ❌ Desorganização
* ❌ Perda de informações
* ❌ Processos manuais

Tudo pensado para trazer praticidade e eficiência no dia a dia escolar.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Supabase (Backend + Banco de Dados)

---

## Como funciona

O sistema utiliza o **Supabase** como backend para gerenciar todos os dados:

* Armazenamento em tempo real
* Integração via API
* Atualizações instantâneas na interface

---

## Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Pontua-Ai/site.git
```

### 2. Acesse a pasta

```bash
cd site
```

### 3. Abra o projeto

Basta abrir o arquivo `index.html` no navegador.
